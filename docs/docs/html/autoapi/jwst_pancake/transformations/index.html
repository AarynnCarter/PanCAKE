jwst_pancake.transformations.align_fourierLSQ(reference, target, mask=None)¶

LSQ optimization with Fourier shift alignment

Parameters:
referencend array

N x K image to be aligned to

targetnd array

N x K image to align to reference

masknd array, optional

N x K image indicating pixels to ignore when performing the minimization. The masks acts as a weighting function in performing the fit.

Returns:
resultslist

[x, y, beta] values from LSQ optimization, where (x, y) are the misalignment of target from reference and beta is the fraction by which the target intensity must be reduced to match the intensity of the reference.
